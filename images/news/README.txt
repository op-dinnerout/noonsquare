  소식 팝업 (News Popup) - 운영 가이드
═══════════════════════════════════════════════════

[1] 새 소식 등록하기 (팝업 켜기)
─────────────────────────────
  1. 이미지 파일을 이 폴더(site/images/news/)에 업로드.
     예: popup.jpg
  2. site/index.html 파일을 열어서 아래 부분을 수정.

     // ── 소식 팝업 설정 ──
     const NEWS_IMAGE = '';           ← 여기에 'images/news/popup.jpg' 입력
     const NEWS_LINK  = '';           ← 클릭 시 이동할 URL (선택, 비워두면 클릭 무반응)

     수정 예시:
     const NEWS_IMAGE = 'images/news/popup.jpg';
     const NEWS_LINK  = 'https://www.instagram.com/noonsquare/';

  3. 커밋(저장) → 자동으로 사이트에 반영됨.


[2] 소식 끄기 (팝업 숨기기 + 메뉴 숨기기)
─────────────────────────────
  index.html 의 NEWS_IMAGE 를 빈 문자열로 변경:

     const NEWS_IMAGE = '';

  자동으로 헤더의 "소식" 메뉴도 사라지고, 팝업도 안 뜸.


[3] 이미지 권장 사양
─────────────────────────────
  ▸ 형식: JPG, PNG, WebP
  ▸ 가로:세로 비율: 4:5 또는 3:4 권장 (세로형)
  ▸ 크기: 가로 600~900px 정도
  ▸ 파일 용량: 300KB 이하 권장 (빠른 로딩)


[4] 팝업 동작 방식
─────────────────────────────
  ▸ 페이지 진입 시 자동으로 팝업이 열림
  ▸ 사용자가 "오늘 하루 보지 않기" 체크 후 닫으면 24시간 동안 안 뜸
  ▸ 일반 닫기(X / 배경 클릭 / ESC) 후엔 헤더의 "소식" 메뉴로 재열기 가능
